# MEMOMed

## Description

Simple GUI desktop application for storing and viewing medical info without need of external services.

## How to use

On login page you might be prompted to create a profile. 
To create a profile you simply input your First and Last names. Profile will be created and any further data or measurements you add will be recorded to this profile. 

Multiple users can use multiple profiles, none of them will be able to see each other records.

## Installation

Installation is not needed! 

You only need to download executable files corresponding to your OS (Linux or Windows) and run it. That's it!
